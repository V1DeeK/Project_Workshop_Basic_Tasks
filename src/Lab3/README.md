# Инструкции по запуску программ из Lab3

## Структура Lab3

Lab3 содержит следующие задания:
- **00** - FizzBuzz (консольная программа)
- **01** - Game Loop (движущийся круг)
- **02** - Moving Circle (круг, движущийся по окружности)
- **03** - Ellipse (статический эллипс)
- **sfml2.1** - Волновое движение с отталкиванием (новое задание)
- **sfml2.2** - Polar Rose (полярная роза)

## Как собрать все программы

1. Откройте терминал (PowerShell) в корневой папке проекта
2. Перейдите в папку Lab3:
   ```powershell
   cd src\Lab3\sfml.2
   ```

3. Создайте папку `build` (если её нет):
   ```powershell
   if (-not (Test-Path build)) { mkdir build }
   ```

4. Перейдите в папку `build`:
   ```powershell
   cd build
   ```

5. Запустите CMake для конфигурации проекта:
   ```powershell
   cmake ..
   ```

6. Соберите все программы:
   ```powershell
   cmake --build .
   ```

   Или соберите конкретную программу:
   ```powershell
   cmake --build . --target 00      # FizzBuzz
   cmake --build . --target 01      # Game Loop
   cmake --build . --target 02      # Moving Circle
   cmake --build . --target 03      # Ellipse
   cmake --build . --target sfml2.1 # Волновое движение с отталкиванием
   cmake --build . --target sfml2.2 # Polar Rose
   ```

## Как запустить программы

После сборки исполняемые файлы будут находиться в папке `build`:

### Консольные программы:
```powershell
# FizzBuzz (консольная программа)
.\00\00.exe
```

### Графические программы (SFML):
```powershell
# Game Loop - движущийся круг
.\01\01.exe

# Moving Circle - круг, движущийся по окружности
.\02\02.exe

# Ellipse - статический эллипс
.\03\03.exe

# Волновое движение с отталкиванием
.\sfml2.1\sfml2.1.exe

# Polar Rose - полярная роза
.\sfml2.2\sfml2.2.exe
```

## Альтернативный способ (из корневой папки проекта)

Если вы находитесь в корневой папке проекта (`Labs 1-4 CPP`):

```powershell
# Перейти в папку build
cd src\Lab3\sfml.2\build

# Собрать все программы
cmake --build .

# Запустить программы
.\00\00.exe
.\01\01.exe
.\02\02.exe
.\03\03.exe
.\sfml2.1\sfml2.1.exe
.\sfml2.2\sfml2.2.exe
```

## Описание программ

- **00 (FizzBuzz)**: Выводит числа от 1 до 100, заменяя кратные 3 на "Fizz", кратные 5 на "Buzz", а кратные 15 на "FizzBuzz"
- **01 (Game Loop)**: Демонстрирует базовый игровой цикл - синий круг движется горизонтально по экрану
- **02 (Moving Circle)**: Круг движется по окружности вокруг центра окна
- **03 (Ellipse)**: Рисует зеленый эллипс в центре окна
- **sfml2.1 (Wave Motion with Bounce)**: Красный шарик движется по синусоиде и отталкивается от левой и правой стенок окна
- **sfml2.2 (Polar Rose)**: Анимированная полярная роза, движущаяся по окружности

## Примечания

- Убедитесь, что SFML установлен и правильно настроен
- Если возникают ошибки компиляции, проверьте пути к SFML в CMakeLists.txt
- Все программы используют SFML 3, но также поддерживают SFML 2 (автоматическое определение)
